# Clio's Hecke Algebra Results: Reference Summary

> Summary of key results from Clio Vega's work on Hecke algebras and Demazure operators, compiled for cross-reference with the signed laxator project.

## 1. Cactus Group Relations at q = 0

The **cactus group** J_n acts on representations of the Hecke algebra H_n(q). At q = 0, Clio established that:

- The Demazure operators satisfy the cactus group relations exactly.
- J_n is generated by involutions s_{p,q} (reversing intervals [p,q]) with specific commutation and nesting relations.
- At q = 0, these generators correspond to specific combinatorial operations on the crystal basis of representations.

This is significant because the cactus group appears naturally in the theory of coboundary monoidal categories (Henriques-Kamnitzer), connecting Clio's work to the categorical framework of the signed laxator.

## 2. Coboundary Monoidal Structure (Not Braided)

A key finding: the relevant monoidal structure on Hecke representations at q -> 0 is **coboundary**, not braided.

- **Braided monoidal:** the braiding sigma_{A,B} : A tensor B -> B tensor A satisfies the Yang-Baxter equation. Path-dependent (different braidings give different results).
- **Coboundary monoidal:** the commutor sigma_{A,B} satisfies sigma_{B,A} . sigma_{A,B} = id (involutory) and the **cactus relation** instead of Yang-Baxter. Path-independent.

**Why this matters for the signed laxator:** A coboundary structure means the "sign" of the laxator depends only on the endpoints (initial and final topological invariants), not on the path of deformation. This is precisely the "flat connection" property -- parallel transport is path-independent. If phi_G depends only on (beta_1, lambda_2) and not on how you got there (which edges were added/removed in which order), the coboundary structure explains why.

## 3. Sharp Transition at q = 0.001

Clio observed a **sharp phase transition** in the behavior of Demazure operators:

- For q > 0.001: the crystal and atom decompositions are "mixed" -- contributions from both bases overlap.
- For q < 0.001: clean separation between crystal (transient/combinatorial) and atom (persistent/algebraic) components.
- The transition is **not smooth** -- it resembles a phase boundary.

**Parallel in evolutionary dynamics:** The NK landscape parameter K plays an analogous role. At K = 0 (smooth landscape), topology has no effect on diversity (eta^2 ~ 0). At K >= 2, topological effects "activate" sharply (C93: eta^2 scales monotonically with K). The signed laxator's prediction: q and K control the same phase transition from different directions.

## 4. Rank = n! / 2^floor(n/2), Independent of q

The **cactus midpoint operator** (the element of J_n that reverses the full interval [1,n]) has:

```
rank = n! / 2^floor(n/2)
```

This rank is **independent of q** -- it is the same for q = 0 and q = 1 (and all values in between).

**Why this matters:** The Tits deformation theorem says H_n(q) is isomorphic to C[S_n] as an algebra for all q != 0. The rank invariance extends this to include q = 0 for this specific operator. This means:

- The **total capacity** of the representation is q-invariant.
- Only the **decomposition** into crystal vs atom components changes with q.
- Analogy: the total diversity capacity of a graph is fixed by |V| and |E|, but the decomposition into transient (beta_1) vs persistent (lambda_2) channels changes with the dynamics.

## 5. Coboundary = Flat Connection

The coboundary structure on the cactus group action yields a **flat connection** on a certain fiber bundle over the configuration space of points.

- **Flat connection:** parallel transport depends only on homotopy class of path, not the path itself.
- **Fiber bundle:** over the space of "topological configurations" (different graph structures), each fiber is the space of possible diversity outcomes.
- **Flat:** the diversity outcome depends only on the topological invariants (beta_1, lambda_2), not on the history of how the graph was constructed.

This is the strongest structural evidence for the signed laxator being well-defined: flatness means phi_G is a function of G's invariants, not of some arbitrary choice of presentation.

## 6. The Mobius/PGL(2) Discovery

Clio's most recent result: the **limiting permuton** (the continuous limit of permutation matrices arising from Hecke representations) depends only on the compound parameter:

```
kappa = (1 - qp) / (1 - p)
```

where q is the Hecke parameter and p is a representation parameter.

### Phase Diagram on the Riemann Sphere

The parameter kappa lives naturally on the **Riemann sphere** C union {infinity} (equivalently, CP^1). The phase diagram of the limiting permuton has:

- Different regions on the Riemann sphere corresponding to qualitatively different limiting behaviors.
- The GA-like regime and LLM-like regime potentially corresponding to different regions of this phase diagram.
- The signed laxator's sign function sigma(beta_1, lambda_2) may be the pullback of a function on the Riemann sphere via kappa.

### PGL(2) Action

The group PGL(2, C) acts on CP^1 by Mobius transformations:

```
kappa |-> (a*kappa + b) / (c*kappa + d)
```

If the limiting permuton is equivariant under some subgroup of PGL(2), this would give a symmetry of the signed laxator -- different-looking parameter regimes that are actually equivalent up to a Mobius transformation.

## 7. Open Questions from Clio

### (a) RP^1 or C union {infinity}?

Is the natural home of kappa the real projective line RP^1 or the full Riemann sphere C union {infinity}?

- If RP^1: the phase diagram is one-dimensional, and there are exactly two "sides" (positive and negative sign). This would give a clean binary sign for the laxator.
- If C union {infinity}: the phase diagram is two-dimensional, and the sign function could be more complex (potentially capturing more than two regimes).

Current evidence leans toward RP^1 for the real-parameter case (q, p real), but the algebraic structure naturally lives over C.

### (b) Fibers Preserved or Mixed?

As kappa moves along a path on the Riemann sphere:

- **Preserved:** each fiber (representation at a given kappa) maps isomorphically to the next. The crystal/atom decomposition "slides" but maintains its structure. This would mean the signed laxator's 2-cells are isomorphisms (not just morphisms).
- **Mixed:** fibers can merge, split, or exchange components. The crystal/atom decomposition genuinely changes topology. This would mean phi_G has non-trivial kernel or cokernel at certain values of kappa.

The flat connection (Section 5) suggests preservation, but this needs careful verification, especially at the singular points of the Riemann sphere (kappa = 0, 1, infinity).
